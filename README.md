# 🥋  Sistema de Cadastro e Contrato de Alunos

Sistema desenvolvido para gerenciamento de cadastro de alunos e geração de contratos da escola de lutas.

## 📋 Sobre o Projeto

Este projeto tem como objetivo facilitar o processo de matrícula de novos alunos, centralizando informações cadastrais e automatizando a formalização contratual.

A aplicação permite o preenchimento de dados pessoais, responsáveis legais (quando necessário), informações de contato e aceite dos termos contratuais da academia.

---

## 🚀 Funcionalidades

- Cadastro de alunos
- Cadastro de responsáveis
- Registro de contatos
- Aceite digital de contrato
- Validação de campos obrigatórios
- Armazenamento de informações
- Consulta de cadastros
- Atualização de dados dos alunos

---

## 📄 Dados Cadastrais

O formulário coleta informações como:

### Dados do Aluno

- Nome completo
- CPF
- Data de nascimento
- Sexo
- Endereço
- Cidade
- Estado
- CEP
- Telefone
- E-mail

### Responsável Legal (Opcional) /////

- Nome completo
- CPF
- Grau de parentesco
- Telefone
- E-mail

### Informações da Modalidade

- Modalidade escolhida
- Data de matrícula ///
- Plano contratado

---

## 📝 Contrato

O contrato possui cláusulas relacionadas a:

- Matrícula do aluno
- Direitos e deveres do aluno
- Normas internas da academia
- Pagamentos e mensalidades
- Cancelamento
- Responsabilidade sobre lesões decorrentes da prática esportiva ///
- Uso de imagem (opcional)
- Assinatura do aluno ou responsável ////
